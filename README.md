## 🚀 Hướng Dẫn Cài Đặt

### Yêu Cầu Môi Trường
- **Node.js**: Phiên bản mới nhất hoặc từ `14.x` trở lên.
- **npm** hoặc **yarn**: Để quản lý các gói thư viện.
- **Expo CLI**: Nếu chưa cài đặt, hãy chạy lệnh:
  ```bash
  npm install -g expo-cli
