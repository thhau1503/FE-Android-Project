**Tên dự án**: App Đặt Phòng Trọ  
**Thành viên phát triển**:

## Ứng dụng này được xây dựng bằng **React Native** và **Expo**, cung cấp giải pháp quản lý đặt phòng trọ hiệu quả với giao diện người dùng thân thiện. Dự án sử dụng nhiều thư viện hỗ trợ như điều hướng, bản đồ, xử lý đa phương tiện và quản lý trạng thái.

---

## 🚀 Hướng Dẫn Cài Đặt

### Yêu Cầu Môi Trường
- **Node.js**: Phiên bản mới nhất hoặc từ `14.x` trở lên.
- **npm** hoặc **yarn**: Để quản lý các gói thư viện.
- **Expo CLI**: Nếu chưa cài đặt, hãy chạy lệnh:

  npm install -g expo-cli
## Clone Dự Án
Clone dự án từ repository:

    git clone https://github.com/thhau1503/FE-Android-Project.git
cd folder-name
## Cài Đặt Các Phụ Thuộc
### Sử dụng npm
    npm install
### Sử dụng yarn
    yarn install
# ▶️ Hướng Dẫn Chạy Ứng Dụng
## Chạy trên Android
    npm run android
### or
    npx expo start
## Chạy trên iOS
    npm run ios
## Chạy trên Web
    npm run web
## Khởi động với Expo CLI
    npm start
